Why this CSS helps performance

  • System font stack avoids extra font downloads.
  • Small stylesheet size, variables, and simple selectors reduce painting cost.
  • Respect for prefers-reduced-motion improves perceived performance.
  • Layer promotion for interactive elements reduces jank.

Tips

  • Inline critical CSS in the HTML head for the fastest first paint.
  • Defer non-critical CSS with rel="preload" -> rel="stylesheet".
  • Use system fonts or font-display: swap to avoid FOIT.
  • Limit heavy shadows and large images above the fold.
